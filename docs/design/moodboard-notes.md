# ムードボード → トークン（メモ）

- **参照**: ジャパンディ系ボード（オフホイト・ライトウッド・自然光）。アクセントは **セージ／スレート** に固定（紫グラデ・ネオンは使わない）。
- **Primary `#5B7268`**: セージ寄りのグリーングレー。CTA・フォーカス・ブランドの軸。
- **Slate `#5E6E78`**: 補助リンク・サブ強調。Primary より青みのニュートラル。
- **背景**: 純白 `#FFF` だけにせず `canvas` / `surface` で温かさを出す。
- **運用者（PC）**: 既存の `gray-*` ユーティリティも使える。新規パーツは `bg-canvas` / `border-border` などセマンティック色で揃えると一貫する。
- **エンドユーザー（スマホ）**: 同トークンを LIFF 側の `:root` CSS 変数にコピーしてよい（`design-tokens.json` が正本）。
